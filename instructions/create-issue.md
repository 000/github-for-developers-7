## How to create a new Issue

1. Click the **Issues** tab.
2. Click **New Issue**.
3. Type an **Issue Title**
4. Type a **Comment** as instructed by the facilitator.
5. Type **@** and the instructor's name to @mention the instructor.
6. Click Submit new issue.
